# Google Chat User Guide

Google Chat is best for Google Workspace organizations that want CoWork available in spaces and direct messages.

## Best For

- Google Workspace teams.
- Shared spaces that need summaries or task intake.
- Direct-message task control.
- Recurring status reports into Google Chat spaces.

## Key Features

- Google Chat app integration.
- Spaces and direct messages.
- Threaded conversations.
- Service-account based setup.
- Shared CoWork gateway lifecycle after messages are delivered.

## Daily Workflow

Use DMs for personal tasks and sensitive work. Use spaces for shared workflows:

```text
/new
Create a concise summary of the blockers discussed here.
```

For scheduled tasks, make the desired final channel output explicit.

## Best Practices

- Use spaces for shared team visibility.
- Use DMs for approvals and private context.
- Keep scheduled outputs short and structured.
- Use `/new` when moving between unrelated space topics.
- Keep app access scoped to intended spaces.

## Watch-Outs

- Google Chat bot support depends on Google Workspace configuration.
- Admin policy may limit where the app can be installed.
- Service account credentials should be treated as sensitive.
